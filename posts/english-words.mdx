### 记录遇到的一些单词

- caller 呼叫者 ；调用者
- callee 被呼叫者；被调用者
- Immutable：不可变的
- circular list 循环链表
- intl abbr. international 的缩写 国际化的 / i18n 也是国际化的缩写
- populate 填充
- interactive adj.交互式， 人机对话的，相互交流的
- get stuck 卡住了
  <Paragraph>
    If you <Strong>get stuck</Strong>, you can reach out to the community on
    GitHub Discussions.
  </Paragraph>
- iterate 迭代
  <Paragraph>
    This will help you <Strong>iterate</Strong> on your app quickly.
  </Paragraph>
- go back and forth 来来回回
  <Paragraph>
    Let's checks to see if it works. You should now have a link on each page,

  <Paragraph>

- client-side navigation 客户端导航
  <Paragraph>
    <Strong>Client-side navigation </Strong>means that the page transition
    happens using Javascript, which is faster than the default navigation done
    by the browser.
  </Paragraph>
  <Paragraph>

- initially 最初
- furthermore 此外 而且

- third-party 第三方
- evolved 进化了的
- illustrate 解释
- hydration 水合（作用）

    page fully interactive.(This process is called <Strong>hydration</Strong>).
  </Paragraph>
- so far 到目前为止
  <Paragraph>
    <Strong>So far</Strong>, all the pages we've created do not require fetching
    external data.
  </Paragraph>
- individual 个人的
- ensential information 基本信息

- wrap up 完成

- passkey 万能钥匙；专用钥匙
